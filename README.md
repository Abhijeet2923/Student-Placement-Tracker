# Student Placement Tracker

A Python GUI application for managing student placement data.

## Features
- Student dashboard
- Company management
- Placement status tracking
- Department-wise records
- MySQL database integration
- Tkinter GUI

## Tech Stack
- Python
- Tkinter
- MySQL
- Matplotlib
- NumPy

## Project Structure

Student-Placement-Tracker
│
├── main.py
├── dbe.py
├── run_gui.py
├── .gitignore
└── README.md

## How to Run

Install dependencies:

pip install mysql-connector-python matplotlib numpy

Run the application:

python main.py
